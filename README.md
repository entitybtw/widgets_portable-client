# widgets_portable-client
pc client for widgets_portable

## versions
- linux(done), available at linux/client.sh
- windows(not done)
- macOS(not done)

## installation
### linux version
1. install 'zenity' package from your distrubution package manager (e.g. arch-based - pacman, debian-based - apt, etc..)
2. download client.sh from linux/ folder
3. make it executable ```chmod +x client.sh```
4. run this command: ```./client.sh --install```
5. done! launch your app from any app-launcher (e.g. rofi, wofi, etc..)
